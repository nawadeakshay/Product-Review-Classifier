PROJECT OVERVIEW
================================================================================

Product Review Classifier is a production-grade machine learning system that
automatically analyzes and categorizes customer product reviews at scale. The
system employs a dual-stage classification pipeline:

  Stage 1: SENTIMENT DETECTION
  ├─ Deep Learning Model (RoBERTa)
  ├─ Binary Classification: Positive vs Negative
  └─ Accuracy: 96.2% (Industry-competitive)

  Stage 2: BUG CATEGORIZATION  
  ├─ Rule-Based Intelligent Classification
  ├─ Multi-class: Audio, Login, UI, Other
  └─ Weighted F1-Score: 0.887 (88.7%)

KEY INNOVATION: Hybrid approach combining deep learning (accuracy) with
rule-based logic (explainability) achieving optimal speed-accuracy-interpretability
tradeoff.

DELIVERABLE: End-to-end pipeline with data preprocessing, model training,
GPU-optimized inference, and automated Excel reporting.

TEXT PREPROCESSING PIPELINE:
  Input:  "Awful app!!! 😡 Visit http://spam.com #trash"
  Step 1 → Lowercase
  Step 2 → Remove URLs (http://...)
  Step 3 → Remove emojis & non-ASCII
  Step 4 → Remove special characters
  Step 5 → Collapse whitespace
  Output: "awful app trash"

WORKFLOW:
  1. Load CSV with review text
  2. Clean & preprocess text
  3. Tokenize using RoBERTa tokenizer (max 128 tokens)
  4. Batch process through sentiment model
  5. For negative reviews: apply bug classification rules
  6. Generate versioned Excel report
  7. Auto-open in Excel for user review
